# qt5-app
这是一个用于嵌入式Linux平台的一整套基础软件基于Qt5.5的widget，主要应用在4.3寸、5寸、7寸的横屏设备上面。你可以交叉编译它们然后拷贝到Linux kernel的ARM、RISC-V平台上面的Buildroot文件系统。
